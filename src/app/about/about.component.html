Vài nét

Đây là chương trình Web App đầu tiên dùng Angular (Single Page Application - SPA)

Bên server thì dùng Fake Json Server (server giả)

Bên client thì dùng ngôn ngữ Angular

Nhiệm vụ của trương trình là quản lý sinh viên đơn giản là thêm, chỉnh sửa, xóa dữ liệu ...

Mục đích là thực tập xem viết 1 web app dùng Angular !
